- Add depth using colour and shadows (使用色彩和陰影增加深度) - 頁 120

### 5. Typography (字體排版) - 頁 229

- Use a single sans serif typeface (使用單一無襯線字體) - 頁 230
- Evoke emotion using a second typeface for headings (使用第二種字體為標題增添情感) - 頁 237
- Use regular and bold font weights only (僅使用標準和粗體字重) - 頁 239
- Use a type scale to set font sizes (使用字體比例設定字級) - 頁 241
- Make long body text bigger (增大長文本字級) - 頁 244
- Use at least 1.5 line height for long body text (長文本使用至少 1.5 行高) - 頁 245
- Decrease line height as font size increases (字級增大時減少行高) - 頁 247
- Ensure ideal line length (確保理想的行長) - 頁 248

- Ensure text on photos is legible (確保照片上的文字清晰可讀) - 頁 254
- Avoid light grey and pure black text (避免淺灰和純黑文字) - 頁 257
- Tutorial - Typography (教學 - 字體排版) - 頁 258

### 6. Copywriting (文案撰寫) - 頁 266

- Be concise (保持簡潔) - 頁 267
- Use sentence case (使用句首大寫) - 頁 269
- Use plain and simple language (使用簡單明瞭的語言) - 頁 270
- Front-load text (重點前置) - 頁 271
- Use the inverted pyramid (使用倒金字塔結構) - 頁 272
- Limit the use of abbreviations and acronyms (限制縮寫使用) - 頁 274
- Limit the use of UPPERCASE (限制全大寫使用) - 頁 275
- Break up content using descriptive headings and bullets (使用描述性標題和項目符號分段) - 頁 276
- Avoid using "my" on form labels (表單標籤避免使用「我的」) - 頁 278
- Use vocabulary consistently (用詞保持一致) - 頁 279
- Use numerals for numbers (數字使用阿拉伯數字) - 頁 281
- Avoid full stops if possible (盡可能避免句號) - 頁 283
- Ensure text length is similar across similar interface elements (相似介面元素的文字長度應相似) - 頁 284
- Ensure text links describe their destination (確保文字連結描述其目的地) - 頁 285
- Write clear error messages (撰寫清晰的錯誤訊息) - 頁 287

### 7. Buttons (按鈕) - 頁 295

- Define 3 buttons weights (定義 3 種按鈕權重) - 頁 296
- Use a single primary button for the most important action (最重要動作使用單一主要按鈕) - 頁 303
- Use secondary buttons for less important actions (次要動作使用次要按鈕) - 頁 305
- Use tertiary buttons for the least important actions (最不重要動作使用第三級按鈕) - 頁 306
- Try to avoid disabled buttons (避免使用禁用按鈕) - 頁 307
- Left align buttons (按鈕左對齊) - 頁 312
- Ensure button text describes the action (確保按鈕文字描述動作) - 頁 317
- Ensure buttons have a sufficient target size (確保按鈕有足夠的點擊目標大小) - 頁 318
- Balance icon and text pairs (平衡圖示與文字配對) - 頁 320
- Add friction to destructive actions (為破壞性動作增加摩擦) - 頁 322

### 8. Forms (表單) - 頁 330

- Stack forms in a single column layout (表單使用單欄堆疊佈局) - 頁 331
- Minimise the number of form fields (最小化表單欄位數量) - 頁 336
- Mark optional fields (標記選填欄位) - 頁 337
- Try to avoid optional fields by using opt-ins (盡量透過選擇性加入避免選填欄位) - 頁 338
- Mark both required and optional fields (同時標記必填和選填欄位) - 頁 339
- Match field width to the intended input (欄位寬度符合預期輸入) - 頁 344
- Stick with conventional form field styles (使用傳統表單欄位樣式) - 頁 346
- Display hints above form fields (在表單欄位上方顯示提示) - 頁 348
- Don't use placeholder text instead of a label (不要用佔位符取代標籤) - 頁 350
- Ensure form field labels are close to their fields (確保表單標籤靠近欄位) - 頁 352
- Try to use radio buttons instead of dropdowns (優先使用單選按鈕而非下拉選單) - 頁 353
- Use an autocomplete instead of a long dropdown (使用自動完成取代長下拉選單) - 頁 354
- Use steppers for numeric fields instead of dropdowns (數字欄位使用步進器而非下拉選單) - 頁 356
- Use a checkbox or toggle switch for 2 options (兩個選項使用核取方塊或切換開關) - 頁 358
- Use positive phrasing for checkboxes (核取方塊使用正面措辭) - 頁 360
- Break up long forms into multiple steps (將長表單拆分為多個步驟) - 頁 361
- Group related fields under headings (相關欄位在標題下分組) - 頁 362
- Ensure form field borders are high contrast (確保表單欄位邊框高對比度) - 頁 363
- Choose your form validation approach (選擇表單驗證方式) - 頁 364
- Chapter summary (章節摘要) - 頁 369
